My Projects

A selection of academic, individual, and team-based engineering projects.

Final Year Project

Intelligent EV Charging Strategy for Weak Distribution Grids with High PV Penetration

MATLAB Fuzzy Logic Optimisation Power Systems

Developed a two-level EV charging coordination system for distribution grids under high solar PV penetration. The central controller manages grid-level constraints — thermal line loading, transformer capacity, and voltage limits. A fuzzy logic secondary controller determines charging priority order among connected EVs based on real-time grid conditions. Voltage profile improved from 0.92 pu to 0.95 pu under coordinated charging versus an uncoordinated scenario.

Hardware Project

Automated Fire Detection & Suppression System

LabVIEW DAQ Card IR Sensor Gas Sensor

Designed and implemented a real-time fire detection and suppression system using LabVIEW with DAQ card hardware integration. IR flame sensors and gas sensors continuously monitor the environment. Upon detection, the system automatically activates a water spray motor and triggers CO₂ release — demonstrating closed-loop hardware control for a safety-critical application.

Power Systems

12-Bus Power Network — Economic Dispatch & N-1 Security Analysis

PSSE Python Load Flow

Modelled a 12-bus power network in PSSE and determined optimal generation dispatch for both peak and off-peak operating conditions. Performed N-1 contingency analysis to verify that the system remains secure under single outage events with no line or transformer overloading. All simulation results were independently cross-verified using Python scripts.

Embedded & IoT

Smart Medibox — ESP32 IoT Environmental Monitor & Alarm System

ESP32 Wokwi Wi-Fi / Node Red Embedded C

Designed and simulated an ESP32-based smart medicine box using the Wokwi online simulation platform. The system continuously monitors ambient temperature and humidity; if readings exceed set thresholds, an audible alarm is triggered locally. Node-RED was used to build the IoT dashboard, enabling a remote user to visualise live sensor data over Wi-Fi — demonstrating a complete IoT pipeline from embedded sensor input through to cloud-accessible monitoring output.
